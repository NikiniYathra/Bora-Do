# Bora Do - Android To-Do List App

Bora Do is a Kotlin-based Android mobile application developed for the CS3082 Mobile Computing assignment. The app allows users to create and manage multiple to-do lists, add/edit/delete items, search lists and items, and store data locally using Room Database.

## Features

- Home screen with developer information
- Create and manage multiple to-do lists
- Add, edit, and delete to-do items
- Search lists and items
- Local data persistence using Room Database
- RecyclerView-based list display
- MVVM-style structure using ViewModel and Repository
- Material Design UI components
